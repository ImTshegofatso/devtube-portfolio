# 🚀 DevTube Portfolio Platform

A modern, interactive developer portfolio inspired by YouTube and TikTok interfaces.  
This platform showcases real-world projects, live demos, and short-form video previews in a clean, responsive UI.

---

## ✨ Overview

DevTube is not just a portfolio — it is a **developer showcase platform** that demonstrates:

- Real working applications instead of static screenshots
- Interactive project previews
- TikTok-style Shorts (video demonstrations)
- Clean, production-ready frontend architecture

---

## 🎯 Key Features

### 🎥 Projects Section

- YouTube-style grid layout
- Live Demo & GitHub links per project (hover actions)
- Real thumbnails (not placeholders)
- Responsive across all devices

---

### ⚡ Shorts (Video Showcase)

- TikTok-style full-screen viewer
- Vertical swipe navigation
- Auto-play for active videos
- Mobile-first optimized design
- Supports `.mp4` videos for real demonstrations

---

### 📱 Responsive Design

- Mobile: 2-column Shorts grid
- Desktop: horizontal scroll layout
- Fully adaptive UI

---

### 👤 About Section

- Personal branding profile
- Skills & experience summary
- Contact information
- GitHub & LinkedIn links

---

### 📄 CV Download + Tracking

- Downloadable CV
- Silent tracking using Formspree
- Email notification on each download
- No visible tracking for users

---

### 🧠 Production Safety Enhancements

- Error boundary protection (prevents crashes)
- Safe rendering (handles missing data)
- Optional chaining for stability
- Graceful fallbacks in UI

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Icons:** Font Awesome
- **State:** React Hooks
- **Tracking:** Formspree
- **Deployment:** Vercel / Netlify
